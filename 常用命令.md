# Baxter 工作区常用命令

## 1. 进入工作区与加载环境

```bash
cd /root/baxter_ws
source devel/setup.bash
```

如果是新开终端，建议先执行：

```bash
cd /root/baxter_ws && source devel/setup.bash
```

## 2. 使用 catkin_tools 编译

全量编译：

```bash
catkin build
```

编译指定包：

```bash
catkin build my_robot_pkg
```

编译并显示更详细输出：

```bash
catkin build --verbose
```

清理某个包后重编：

```bash
catkin clean my_robot_pkg -y
catkin build my_robot_pkg
```

## 3. ROS 基础命令

查看当前 ROS 包路径：

```bash
echo $ROS_PACKAGE_PATH
```

查看可用话题：

```bash
rostopic list
```

查看话题频率：

```bash
rostopic hz /topic_name
```

查看节点列表：

```bash
rosnode list
```

查看参数：

```bash
rosparam list
```

## 4. 启动与运行

启动 roscore：

```bash
roscore
```

运行单个 Python 节点：

```bash
rosrun my_robot_pkg preset_executor.py
```

启动 launch 文件：

```bash
roslaunch my_robot_pkg xxx.launch
```

## 5. Baxter 相关

加载 Baxter 环境脚本（按项目实际参数修改）：

```bash
source /root/baxter_ws/baxter.sh
```

查看 Baxter 相关话题（快速筛选）：

```bash
rostopic list | grep baxter
```

## 6. 调试与日志

查看节点信息：

```bash
rosnode info /node_name
```

查看话题详细信息：

```bash
rostopic info /topic_name
```

查看最近日志目录：

```bash
roscd log
```

实时看系统日志（示例）：

```bash
tail -f ~/.ros/log/latest/rosout.log
```

## 7. 常见排查

检查 Python 节点语法：

```bash
python3 -m py_compile src/my_robot_pkg/scripts/preset_executor.py
```

查看可执行权限：

```bash
ls -l src/my_robot_pkg/scripts/preset_executor.py
chmod +x src/my_robot_pkg/scripts/preset_executor.py
```

检查某个包是否被识别：

```bash
rospack find my_robot_pkg
```

## 8. 结束进程（杀死进程）

先查 PID 再结束（推荐）：

```bash
ps -ef | grep 进程名
kill PID
```

强制结束进程（无法正常退出时）：

```bash
kill -9 PID
```

按进程名结束（谨慎使用，可能匹配多个）：

```bash
pkill -f 进程名
```

按端口结束进程（例如占用 11311）：

```bash
fuser -k 11311/tcp
```

ROS 常用：结束所有 ROS 相关进程：

```bash
rosnode kill -a
killall -9 rosmaster rosout gzserver gzclient rviz
```
